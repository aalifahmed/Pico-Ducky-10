Pico Ducky

Turn your Raspberry Pi Pico into a USB Rubber Ducky

Pico Ducky allows you to transform your Pico into a programmable USB HID device using CircuitPython. Follow these steps carefully to set it up.

Installation Steps

1

Download & Extract

Download the Pico Ducky 10.zip and extract it to a folder on your PC.

2

Enter BOOTSEL Mode

Plug your Pico while holding the BOOTSEL button. It will appear as RPI-RP2.

3

Flash CircuitPython

Copy the .uf2 file to the root of RPI-RP2. Pico will reboot as CIRCUITPY.

4

Extract Libraries

Extract adafruit-circuitpython-bundle-10.x-mpy-YYYYMMDD.zip.

5

Copy adafruit_hid

Navigate to lib in the bundle and copy adafruit_hid to CIRCUITPY/lib.

6

Copy Other Libraries

Copy adafruit_debouncer.mpy, adafruit_ticks.mpy, asyncio, and adafruit_wsgi to CIRCUITPY/lib.

7

Copy Pico Ducky Files

Copy boot.py, duckyinpython.py, code.py, pins.py, webapp.py, wsgiserver.py to CIRCUITPY root.

8

Add Payload

Create a DuckyScript file and save it as payload.dd in the root of CIRCUITPY.
